# StarWars.com Notes

Purpose: capture the distinct visual traits of each website-reference screenshot so the current site can later borrow from these eras intentionally instead of vaguely. Dates are used as the unique identifiers.

## How To Use These Notes
- Treat each dated snapshot as an aesthetic reference, not a literal layout blueprint.
- Pull from the notes in layers: palette, typography, navigation chrome, card treatment, and hero composition.
- For a future user-selectable aesthetic switcher, the most sensible approach is to group these screenshots into era families rather than making 17 totally separate themes.

## Distinct Screenshot Notes

### 1997-01-13
- Overall mood: very early web, black void background, sparse composition, almost homemade terminal-map feel.
- Palette: black, white, neon green link accents, thin gray rules.
- Typography: serif body copy with plain system-era rendering; headings feel utilitarian rather than branded.
- Layout: narrow centered content column with large empty space; modular blocks arranged like a rough galactic map.
- UI traits: thin borders, tiny image placeholders, text-heavy navigation, minimal image polish.
- Reusable cues: black starfield-like negative space, green data-link accents, map/coordinate framing, low-fi briefing aesthetic.

### 1997-07-03
- Overall mood: cinematic forest backdrop with overlaid editorial text; much more atmospheric than the January version.
- Palette: forest greens, bark browns, muted grays, white text.
- Typography: condensed all-caps section labels paired with serif paragraph text.
- Layout: full-bleed photo background with vertically stacked link columns over the scene.
- UI traits: content feels embedded directly into photography rather than boxed into cards.
- Reusable cues: environmental photo as page skin, centered logo lockup, white overlay text, location-based link clusters.

### 1998-02-15
- Overall mood: magazine-like and cleaner, moving away from black-space roughness toward editorial curation.
- Palette: soft gray page background, black text, muted bronze/brown link accents.
- Typography: bold outlined site logo, mostly plain sans-serif content, underlined inline links.
- Layout: clean grid of feature blurbs with thumbnail art blocks and generous whitespace.
- UI traits: minimal chrome, article-card thinking without fully formed cards.
- Reusable cues: light neutral background, editorial modules, understated separators, clean archive/museum feel.

### 1998-12-02
- Overall mood: Episode I hype era, black background with metallic logo and blue link treatments.
- Palette: black, chrome silver, electric blue, faint gray outlines.
- Typography: large metallic Star Wars masthead, compact sans-serif body text.
- Layout: asymmetrical homepage with promo blocks floating around a large hero/logo cluster.
- UI traits: thin outlined content panels, franchise-era tabs, promo banners for books/video/features.
- Reusable cues: chrome logo treatment, deep black canvas, blue-linked information nodes, prequel teaser energy.

### 2000-02-29
- Overall mood: parchment-and-sand prequel portal with collectible-card styling.
- Palette: tan parchment, sandy beige, blue-gray module borders, gold logo tones.
- Typography: sci-fi display logo paired with compact sans-serif body text.
- Layout: left content rail with stacked feature strips; right sidebar for sign-in, poll, and category links.
- UI traits: bevelled module edges, episode tabs, boxed poll, thin framed thumbnails.
- Reusable cues: sandstone texture, blue metal trim, Episode I dossier styling, side-rail utility stack.

### 2001-06-29
- Overall mood: transitional early-2000s portal with a large character-driven hero and rounded gray content pods.
- Palette: pale gray background, black navigation, muted greens, gunmetal gray.
- Typography: bold all-caps display for the logo; light sans-serif elsewhere.
- Layout: huge left hero image, compact right column of rounded-corner story panels, very airy lower half.
- UI traits: floating capsule modules, soft shadows/gradients, understated top nav.
- Reusable cues: character-led hero crop, translucent top navigation, rounded story pills, soft atmospheric fade-outs.

### 2002-08-27
- Overall mood: dark, hot, and dramatic Revenge-of-the-Clone-Wars-era page with strong orange glow.
- Palette: black, burnt orange, amber, deep brown, lightsaber white.
- Typography: heavy logo mark with compact uppercase nav.
- Layout: large left hero art, right sidebar news list, thin horizontal nav bars above and below.
- UI traits: edge-lit side panels, glow gradients, sparse but high-contrast information layout.
- Reusable cues: sunset duel palette, strong hero silhouette, thin orange dividers, dramatic side-column news feed.

### 2004-02-24

- Palette: slate blue background, black panels, icy blue highlights, white text.

- Layout: dense tiled dashboard of databank, message boards, shop, webcam, galleries, and promos.

### 2006-02-09

### 2009-04-02

### 2015-04-30

- Palette: black header shell, deep blue starfield, bright yellow CTA, white text, colorful promo art.

- Reusable cues: black shell plus vibrant promo slab, yellow CTA buttons, starfield texture behind modules, strong merchandising emphasis.

## Era Family Suggestions

### 1. Proto-Web Archive
Dates:
- 1997-01-13
- 1997-07-03
- 1998-02-15

Traits:
- sparse layouts
- primitive hyperlink styling
- black or light neutral backgrounds
- text-heavy editorial emphasis

Useful adaptation:
- could become a minimalist “Holonet Archive” theme with serif text, thin rules, and low-fi terminal accents.

### 2. Prequel Portal
Dates:
- 1998-12-02
- 2000-02-29
- 2001-06-29
- 2002-08-27

Traits:
- metallic logos
- sand/chrome/black palettes
- strong promo art
- sidebar-heavy portal structure

Useful adaptation:
- ideal for a Naboo/Coruscant-era theme with metallic headers, parchment panels, and episode-tab navigation.

### 3. Lucasfilm Dashboard
Dates:
- 2004-02-24
- 2006-02-09

Traits:
- many boxed modules
- angled sci-fi panel edges
- databank/control-room feeling
- utility-first homepage density

Useful adaptation:
- strong fit for a “command center” timeline mode with dense widgets, beveled frames, and dashboard stats.

### 4. Clone Wars Broadcast
Dates:
- 2009-04-02
- 2011-10-01
- 2013-04-01

Traits:
- TV-first presentation
- blue neon accents
- hero-slider structure
- cleaner modern content cards

Useful adaptation:
- good basis for an animated-series theme with holographic blues, full-bleed heroes, and strong episode rails.

### 5. Modern Editorial / Streaming
Dates:
- 2015-01-23
- 2015-04-30
- 2019-08-31
- 2019-12-01
- 2026-03-10

Traits:
- dark shell
- category icon nav
- premium campaign hero modules
- streaming/e-commerce crossover

Useful adaptation:
- easiest to adapt to the current site because it already aligns with modern responsive card systems and large hero storytelling.

## Practical Styling Directions For The Timeline
- Use 1997-01-13 as reference for a “Holonet terminal” mode: black background, thin outlines, green links, map-like content modules.
- Use 2000-02-29 as reference for a “Prequel archive” mode: parchment texture, blue-gray borders, stacked sidebar utilities.
- Use 2004-02-24 as reference for a “Databank control room” mode: angular panel corners, modular dashboard cards, steel-blue accents.
- Use 2011-10-01 as reference for a “Clone Wars cinematic” mode: fullscreen hero imagery, translucent black overlays, green/blue saber glows.
- Use 2015-01-23 or 2019-12-01 as reference for a “Modern StarWars.com” mode: dark shell, structured content rails, stronger media cards.
- Use 2026-03-10 as reference for a “Disney-era promo” mode: blue starfield slab backgrounds, bright yellow CTAs, polished marketing collage cards.

## Recommendation For A Future Aesthetic Selector
Start with 4 modes instead of 17:
- Holonet Archive
- Prequel Portal
- Databank Dashboard
- Modern StarWars.com

Reason:
- these modes are visually distinct enough to feel meaningful
- they cover the strongest shifts in the site’s design history
- they are implementable without turning the CSS into an unmaintainable museum of one-off themes
